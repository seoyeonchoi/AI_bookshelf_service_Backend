# medic-backend

## 사용 스택

1. Express
2. Node.js (v16.15.1)
3. mongoDB
4. nodemon

## 실행 방법

1. `yarn start`: 로컬 서버 실행
2. `yarn dev`: 로컬 테스트 서버 실행(예정)

## 주의 사항

1. .gitignore에 저장되지 않는 항목이 있습니다.
2. .gitignore 추가시 파일에 있는 링크로 추가하시길 바랍니다.

## 목적

1. 프로젝트 시작시 기본적으로 갖추어야할 js web server를 만들기 위해서 입니다.

## 배포방법

1. main 브랜치에 커밋하고 원격 저장소에 푸시하기
2. `cd ~/js_server`
3. `git pull`
4. `yarn` 실행하기
